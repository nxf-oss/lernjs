# Dokumentasi Website

## Selamat Datang

Ini adalah dokumentasi utama untuk website kami. Dokumentasi ini mencakup semua aspek dari website termasuk halaman-halaman utama, API, dan kebijakan.

## Struktur Website

Website kami terdiri dari beberapa bagian utama:

```{toctree}
:maxdepth: 2
:caption: Contoh Halaman
:glob:

_example/*
_example/_api/*
```

### Halaman Utama

- Homepage - Halaman utama website
- Contact - Halaman kontak dan informasi
- Legal Pages - Halaman kebijakan dan persyaratan

### API Endpoints

Kami menyediakan berbagai endpoint API untuk integrasi:

- GET - Untuk mengambil data
- POST - Untuk membuat data baru
- PUT - Untuk mengupdate data
- PATCH - Untuk partial update
- DELETE - Untuk menghapus data

### Dokumentasi

Semua dokumentasi tersedia dalam format yang mudah dipahami dengan contoh penggunaan.

## Fitur Utama

1. **Responsive Design** - Website dapat diakses dari berbagai perangkat
2. **API Integration** - Menyediakan API untuk developer
3. **Security** - Keamanan data pengguna menjadi prioritas
4. **Performance** - Optimasi untuk kecepatan loading
5. **Accessibility** - Dapat diakses oleh semua pengguna

## Cara Menggunakan

### Untuk Pengguna Biasa

1. Kunjungi halaman utama
2. Gunakan menu navigasi untuk menjelajahi konten
3. Hubungi kami melalui halaman contact untuk pertanyaan

### Untuk Developer

1. Baca dokumentasi API
2. Gunakan endpoint yang tersedia
3. Ikuti panduan autentikasi
4. Test dengan environment development terlebih dahulu
